### Sequence to sequence model
Sequence to sequence model is model that transforms sequence of objects and converts them to another sequence of object. For example translation engine, that that translate words from one language to another.

Model contains encoder and decoder. Encoder processes input element by element and stores it in context (some sort of information bank). Encoder sends context to decoder, which decodes by element by element.
