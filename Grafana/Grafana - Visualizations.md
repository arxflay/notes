##### 1. Gauge charts
- This type of visualization is used for monitoring single value, visualizing how much current value differs from minimum and maximum but not limited to two edges (for example different colors for 25%, 50%, 75% and 100%)
- Could also be represented in bar form (bar gauge chart)
- Configuration tutorial https://youtu.be/QwXj3y_YpnE
![[gauge.png]]
##### 2. State
* Similar to timeline chart but has different purpose 
* Used for showing state change of activity (or multiple activities) in specific period of time.
* Example: switch was ON from 13:40 to 15:30 and then switched to OFF

![[state_timeline.png]]
##### 3. Stat
* Chart for monitoring current single value with optional graph

##### 7. Geomap
* This type of visualization is used to show activity in different regions
* Data are GPS location or exact address (with country, state etc.)

* This type of visualization is used to show correlation (how much values are related) between X and Y
